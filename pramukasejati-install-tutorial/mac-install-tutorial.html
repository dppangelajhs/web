How to Install (MacOS )

1. Unduh Pramuka Sejati

Unduh file Pramuka Sejati dari website resmi kami https://s.id/dppangelajhs. Pastikan Anda memilih versi rilis resmi MacOS yang terbaru. Pastikan pula Anda memiliki koneksi internet untuk mengunduh file game.

2. Temukan aplikasi pada Finder

Temukan aplikasi Pramuka Sejati pada Finder. Bila mengalami kesulitan, Anda juga dapat menggunakan Spotlight pada navigasi MacOS dan cobalah buka aplikasi tersebut.

3. Kondisi aplikasi diblokir oleh Gatekeeper

Pada MacOS, terdapat Gatekeeper yang merupakan fitur keamanan sistem operasi MacOS oleh Apple. Ini memberlakukan penandatanganan kode dan memverifikasi aplikasi yang diunduh sebelum mengizinkannya berjalan, sehingga mengurangi kemungkinan menjalankan malware secara tidak sengaja. Secara default, Gatekeeper tidak akan mengizinkan aplikasi yang dianggap "asing" oleh Apple.

Anda perlu melakukan beberapa langkah tambahan agar aplikasi Pramuka Sejati tidak diblokir oleh Gatekeeper. Kami bertanggungjawab dan menjamin keamanan sistem operasi Anda serta menjamin bahwa aplikasi yang diunduh 100% aman dari virus/malware selama Anda mengunduh versi rilis resmi dari website resmi kami https://s.id/dppangelajhs sesuai dengan panduan yang berlaku.

4. Pergi ke pengaturan sistem

Pergi ke pengaturan sistem (System Settings). Secara default, biasanya terdapat pada bagian Dock MacOS. Anda juga bisa pergi ke pengaturan sistem melalui Spotlight di bagian navigasi MacOS.

5. Temukan pengaturan privasi dan keamanan (Privacy & Security)

Pada pencaharian pengaturan, ketik "Privacy & Security" untuk menemukan pengaturan privasi dan keamanan dengan cepat.

6. Klik "Open Anyway" pada Pramuka Sejati

Scroll ke bawah pada pengaturan privasi & kemanan. Lalu temukan aplikasi yang terblokir pada bagian "Security". Pada aplikasi Pramuka Sejati, klik "Open Anyway" untuk menjalankan aplikasi.

7. Konfirmasi pilihan

Klik sekali lagi pilihan "Open Anyway" sebagai bentuk konfirmasi. Pada umumnya, Anda akan diminta memasukkan kata sandi/sidik jari sebagai bentuk keamanan.

8. Buka kembali aplikasi & selamat bermain!

Setelah melakukan semua prosedur diatas, buka kembali aplikasi Pramuka Sejati melalui finder atau spotlight. Seharusnya Anda sudah bisa memainkan aplikasi Pramuka Sejati dengan lancar! Apabila terdapat beberapa kendala, Anda dapat menghubungi kami melalui website ini (klik disini).
