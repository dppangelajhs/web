How to Install (Windows )

1. Unduh Pramuka Sejati

Unduh file Pramuka Sejati dari website resmi kami https://s.id/dppangelajhs. Pastikan Anda memilih versi rilis resmi Windows yang terbaru. Anda dapat menginstall Pramuka Sejati untuk Windows dalam bentuk Executable Format (.exe) ataupun dalam bentuk Arsip (.zip). Pastikan pula Anda memiliki koneksi internet untuk mengunduh file game.

2. Buka aplikasi Pramuka Sejati

Temukan aplikasi Pramuka Sejati pada File Explorer, dan pastikan Anda sudah menemukan file dalam bentuk executable (.exe). Apabila file yang diunduh adalah dalam bentuk arsip (.zip), pastikan Anda telah mengekstraknya lebih dulu. Ketika Anda telah menemukan file aplikasi Pramuka Sejati dalam bentuk (.exe), Anda dapat segera membukanya.

3. Kondisi aplikasi diblokir oleh Microsoft SmartScreen

SmartScreen adalah komponen anti-phishing dan anti-malware berbasis cloud yang disertakan dalam beberapa produk Microsoft. Dalam hal ini, Microsoft SmartScreen mendeteksi aplikasi Pramuka Sejati sebagai aplikasi yang tidak dikenal (dianggap "asing").

Anda perlu melakukan beberapa langkah tambahan agar aplikasi Pramuka Sejati tidak diblokir oleh Microsoft SmartScreen. Kami bertanggungjawab dan menjamin keamanan sistem operasi Anda serta menjamin bahwa aplikasi yang diunduh 100% aman dari virus/malware selama Anda mengunduh versi rilis resmi dari website resmi kami https://s.id/dppangelajhs sesuai dengan panduan yang berlaku. Dalam mengatasi hal ini, Anda dapat klik "More info" pada notifikasi Microsoft SmartScreen.

4. Klik Run Anyway untuk menjalankan aplikasi

Setelah memilih pilihan "More info" tombol bertuliskan "Run anyway" akan muncul untuk memberikan pilihan lebih lanjut kepada pengguna bila pengguna benar-benar mempercayai pihak developer aplikasi tersebut. Untuk dapat menjalankan aplikasi Pramuka Sejati, klik tombol "Run anyway".

5. Selamat bermain!

Sistem operasi Anda akan secara otomatis menjalankan aplikasi Pramuka Sejati dan artinya Anda sudah bisa memainkan aplikasi Pramuka Sejati dengan lancar! Apabila terdapat beberapa kendala, Anda dapat menghubungi kami melalui website ini (klik disini).
